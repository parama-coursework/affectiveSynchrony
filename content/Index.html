ANALYSIS OF SYNCHRONY IN AFFECT AS MEASURED BY PHYSIOLOGICAL AROUSAL IN IN-SITU EDA MEASUREMENTS

This project seeks to utilize data visualization in order to investigate synchrony in affect between occupational therapists and children with Sensory Processing Disorder (SPD) using Electrodermal Activity (EDA) Measurements obtained in-situ, during the childrens therapy sessions. We establish a set of possible measures of synchrony to be visualized, and then compare signa ls of children and care providers over a set of sessions. Comparisons were also done over different sessions with the same care provider, as well as over samples where the care provider was actively interacting with a child versus when they were not. Visualizations explored filtered and unfiltered data, correlation between filtered data samples, average delay between peaking, and average number of peaks.
