VISUALIZATIONS

Below, you will find links to interactive visualizations that display data pertinent to the investigations conducted. Directions are organized by Investigation. All visualizations were constructed using the javascript library D3 created by the Stanford Visualization Group along with an enhanced version of the Rickshaw.js plugin that was customized for the purposes of this set of visualizations. The data that they call are in JSON format.

SESSION PLOTS

For each session plot, you will see a plot of filtered and raw EDA's of the child and occupational therapist on the bottom of the screen over the course of a session. Other plots will indicate correlation between filtered data, and average number of peaks during the selected subset of time.

  • Zooming In: To select a subset of time during the session, drag the bottom sliders in order to stretch the time series to the set of times you are interested in.
  • Selecting Which Time Series to View: Hit the check marks next to the time series legend if you would like to view only filtered or unfiltered data.
  • Switching Between Different Types of Visualizations: In order to view different types of visualizations, one can move from the stacked area chart to the other types of charts, but in order to move backwards to different type of plot, one must refresh the page before selecting the next type of plot.
  • Reordering Time Series, or Moving Time Series In Front or Behind One Another: In order to move one time series in front of another or behind another, simply click the legend's colored squares up or down corresponding to the order in which you would like them to be rendered on top of one another.

CORRELATION PLOTS

These plots show both cumulative correlation over time as well as a moving window correlation over a specified window. I have varied the moving windows for sample sets, with some sample sets showing a moving window of 30 seconds, and some showing a moving window of 120 seconds, in order to demonstrate the differences that yeilds in terms of visual effect of data and ease of detection of visual trends.

DATA CONVERSION

I recieved the initial dataset in excel format. I used a python script written by Peter Combs (pcombs@gmail.com) to convert my excel files into .mat files, and wrote my own conversion scripts to convert my matlab data into the JSON files that are read by D3.
